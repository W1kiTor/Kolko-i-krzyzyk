Nazwa projektu: Kółko i krzyżyk - webpage game

Podstawowe założenia projektu:
1. Działająca gra dla conajmniej jednej osoby.
2. Gra musi zostać stworzona w html css oraz js.
3. Gra musi być prototypem.
4. Musi zawierać podstawowy styl.

Technologie
- HTML
- CSS
- JavaScript
- GitHub (Issues, Milestones, Branches)

Hierarchia projektu
- index.html - Podstawa projektu, Przesyłanie użytkownika do welcome-page.html
- welcome-page.html - Strona przywitania użytkownika oraz start gry
- game.html - Główne okno rozgrywki
- game.js - Logika gry dla game.html

- style.css - Ogólna stylistyka i wygląd dla game.html, welcome-page.html, index.html
