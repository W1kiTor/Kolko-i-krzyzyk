##
# Dokumentacja Projektu - Gra webowa kółko i krzyżyk - ZCMP

Tic-Tac-Future — Gra „Kółko i Krzyżyk” Online
##
# Opis projektu
Jest to strona w przeglądarce z możliwością grania w grę "Kółko i krzyżk" w dwie osoby na jednym użądzeniu. Celem projektu było zapoznanie się z środowiskiem GitHub oraz przygotowanie dowolnego projektu w ramach zajęć z przemiotu "Zarządzania cyfrowym miejscem pracy".

##
# Wykorzystane technologie 

- HTML5
- CSS
- JavaScript
- GitHub

##
# Struktura

- **index.html** - Przesłanie użytkownika do welcome-page.html (Przestrzeń dla dalszego rozwoju)
- **welcome-page.html** - Przywitanie użytkownika oraz opcja rozpoczęcia gry (Przesłanie użytkownika do game.html)
- **game.html** - Glówna strona rozgrywki
- **game.js** - Logika rozgrywki w game.html
- **style.css** - Globalny styl strony
